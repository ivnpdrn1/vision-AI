# El cambio importante en drones

En supermercados:
la cámara es fija.

En drones:
la cámara se mueve constantemente.

Entonces aparecen nuevos problemas:

| Problema     | Ejemplo            |
| ------------ | ------------------ |
| Vibración    | imagen borrosa     |
| Movimiento   | tracking complejo  |
| Altura       | objetos pequeños   |
| Luz variable | sol/nubes          |
| Latencia     | decisiones rápidas |
| Batería      | hardware eficiente |
