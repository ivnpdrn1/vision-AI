When trying to restore the .yml file, I'm getting an error that some libraries could not be found. What should I do?

Sometimes when using certain versions of Operating Systems, conda doesn't need to download a few supporting libraries. This means that those libraries don't exist for those operating systems. If you get this error, its a simple fix! Simply open the .yml file in a text editor, and remove/delete any lines of the libraries that could not be found for your OS.
